# 06 资讯简报助手：AI Agent 领域日报 + 飞书推送

本期视频说明 OpenClaw 为你省下第一波时间的第一个自动化场景应用，完成两类需求：**即时检索并推送简报**，以及**每天固定时间自动执行同一套流程**（定时任务）。核心产出是一份**结构化简报**：覆盖权威媒体、社区论坛与行业报告中的当日要点，并落到可执行动作。
